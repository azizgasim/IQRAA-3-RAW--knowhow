## نظرة عامة | Overview

هذا المستودع يحتوي على أداتين أساسيتين من منصة إقرأ للبحث في التراث الإسلامي:

1. **[المفكرة الذكية](./smart-notebook/)** - نظام تدوين الملاحظات الشخصي للباحث
2. **[زوتيرو الذكي](./zotero-smart/)** - نظام إدارة المراجع العلمية

---

## 📝 المفكرة الذكية | Smart Notebook

> "ليست عقلاً بل وعياً بالعقل"

المفكرة الذكية الشخصية للباحث - طبقة ميتا-معرفية تراقب وتفهم أنماط تفكير الباحث.

### المميزات
- ✅ تدوين الملاحظات مع Markdown
- ✅ ربط الملاحظات ببعضها [[wiki-links]]
- ✅ المشاريع البحثية ورحلة المشروع
- ✅ المرآة المعرفية والزخم البحثي
- ✅ البحث النصي الكامل
- ✅ المراجعة المتباعدة

### الألوان
- **الأساسي**: أزرق (#3b82f6)
- **الثانوي**: بنفسجي (#a855f7)

### التشغيل
```bash
cd smart-notebook

# Backend
cd backend && pip install -r requirements.txt
uvicorn main:app --port 8001 --reload

# Frontend
cd frontend && npm install && npm run dev
# يعمل على المنفذ 3001
```

## 📚 زوتيرو الذكي | Zotero Smart

نظام متكامل لإدارة المراجع العلمية، مصمم للباحثين في التراث الإسلامي.

### المميزات
- ✅ إدارة المراجع (كتب، مقالات، رسائل، ...)
- ✅ مجموعات هرمية ووسوم
- ✅ 7 أنماط اقتباس (APA, MLA, Chicago, Harvard, IEEE, Vancouver, شيكاغو العربي)
- ✅ استيراد/تصدير BibTeX, RIS, JSON
- ✅ البحث النصي الكامل
- ✅ التعليقات والملاحظات على المراجع

### الألوان
- **الأساسي**: أخضر (#10b981)
- **الثانوي**: كحلي (#1e3a5f)

### التشغيل
```bash
cd zotero-smart

# Backend
cd backend && pip install -r requirements.txt
uvicorn main:app --port 8002 --reload

# Frontend
cd frontend && npm install && npm run dev
# يعمل على المنفذ 3002
```

## التكامل | Integration

الأداتان متكاملتان مع بعضهما ومع أدوات منصة إقرأ الأخرى:

```
┌─────────────────────────────────────────────────────────────┐
│                      منصة إقرأ                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ┌──────────────────┐      ┌──────────────────┐            │
│  │ المفكرة الذكية   │◄────►│  زوتيرو الذكي   │            │
│  │  :3001 / :8001   │      │  :3002 / :8002  │            │
│  └────────┬─────────┘      └────────┬─────────┘            │
│           │                         │                       │
│           ▼                         ▼                       │
│  ┌──────────────────────────────────────────────┐          │
│  │              UltraGraph                       │          │
│  │            خريطة المعرفة                      │          │
│  └──────────────────────────────────────────────┘          │
│                         ▲                                   │
│                         │                                   │
│  ┌──────────────────────┴───────────────────────┐          │
│  │                                               │          │
│  │    القارئ الذكي  ◄────►  الوكيل الذكي       │          │
│  │                                               │          │
│  └───────────────────────────────────────────────┘          │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### نقاط التكامل

| من | إلى | الوظيفة |
|---|---|---|
| المفكرة | UltraGraph | عرض الملاحظات في خريطة المعرفة |
| المفكرة | زوتيرو | إضافة مرجع من ملاحظة |
| المفكرة | الوكيل | إرسال سؤال للوكيل |
| زوتيرو | المفكرة | إرسال مرجع للملاحظات |
| زوتيرو | UltraGraph | عرض المراجع في الخريطة |
| القارئ | المفكرة | إرسال اقتباس |
| القارئ | زوتيرو | إضافة مرجع |
| الوكيل | المفكرة | إرسال إجابة |
| الوكيل | زوتيرو | اقتراح مراجع |
