## 📋 فهرس الحزم

### الحزمة 7: عائلة التطهير (Purification Family)
**الملف:** `pkg7_purification/purification_family.yaml`

| الوكيل | الوظيفة |
|--------|---------|
| ① الحراسة الشرعية والفطرية | البوابة الأخيرة - حراسة الثوابت |
| ② الخطر البريء | كشف الأخطاء التقنية (هلوسة، تناقض) |
| ③ التحيز المصمَّم | كشف قرارات التصميم المقصودة |
| ④ الهيمنة الثقافية | كشف التحيزات الثقافية في البيانات |
| ⑤ المراقب الصامت | Meta-monitor يراقب المراقبين |

---

### الحزمة 8: طبقة المستشارين (Advisors Layer)
**المجلد:** `pkg8_advisors/`

| الملف | الوكيل | الوظيفة |
|-------|--------|---------|
| `01_observatory.yaml` | المرصد | رصد النماذج والأدوات والأسعار |
| `02_model_router.yaml` | وكيل اختيار النموذج | توجيه المهام للنموذج الأنسب |
| `03_shura_protocol.yaml` | بروتوكول الشورى | استشارة عدة نماذج للقرارات الحساسة |

---

### الحزمة 9: وكلاء الأدوات (Tool Agents)
**المجلد:** `pkg9_tools/`

| الملف | الوكيل | الوظيفة |
|-------|--------|---------|
| `01_zotero_agent.yaml` | وكيل زوتيرو | إدارة المراجع باللغة الطبيعية |
| `02_smart_notebook_agent.yaml` | المفكرة الذكية | القرين الفكري + تكامل أوبسيديان |

---

### الحزمة 10: وكلاء الدعم البحثي (Research Support)
**المجلد:** `pkg10_research_support/`

| الملف | الوكيل | الوظيفة |
|-------|--------|---------|
| `01_methodology_agent.yaml` | وكيل المناهج العلمية | استحضار المنهجيات، كشف الأيديولوجيات |
| `02_academic_support_agent.yaml` | وكيل الدعم الأكاديمي | خبير التقاليد الجامعية |
| `03_academic_companion.yaml` | القرين الأكاديمي | رفيق الحوار والنقاش |

---

### الحزمة 11: إدارة الوكلاء (Agent Management)
**المجلد:** `pkg11_agent_manager/`

| الملف | الوكيل | الوظيفة |
|-------|--------|---------|
| `01_agent_manager.yaml` | مدير الوكلاء | التحكم باللغة الطبيعية + سجل المطالبات |
